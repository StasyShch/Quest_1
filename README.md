# Narnia Quest Web Page - interaction with The Bird

A simple web page for a children's quest game inspired by **The Lion, the Witch and the Wardrobe**.

## Features

* Embedded YouTube video with a game clue.
* Password input field.
* Password validation button.
* Success message displayed when the correct password is entered.
* Hidden clue revealed after successful validation.

## How It Works

### Step 1: Watch the Video

Players watch the embedded video and listen carefully to the clue.

### Step 2: Enter the Password

Players type the discovered password into the input field.

### Step 3: Check the Password

Press the **Check** button.

### Step 4: Reveal the Next Clue

If the password is correct, the next clue or instruction is displayed.
